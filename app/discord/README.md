Help students create private channels for their teams and assign them to it on request.

### MS Bot
This bot is a private bot installed only on our Discord server, to add new functions, add them to the python file and run it
These are the commands available:
 - `clear_main` deletes all messages in the main chat, run it before or after an activity
 - `kick_students` kicks out all of the students, use this after an activity has ended
 - `kickall` kicks out ALL users, including other admins

Moreover two hooks are present to assign roles upon joining based on the URLs above
